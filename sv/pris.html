Vanliga frågor

  • Kan jag testa innan jag prenumererar?

    Som en gratisanvändare har du begränsad tillgång till alla våra verktyg. Genom att uppgradera till Premium får du tillgång till samma verktyg med de högsta gränserna för antal uppgifter och filstorlekar.

  • Jag vill köpa flera användarkonton. Kan jag få rabatt?

    iLovePDF har redan ett mycket konkurrenskraftigt pris. Upp till 10 användare har vi bara rabatt på årsmedlemskap, på 33%. För större volymer användarkonton, vänligen mejla oss.

  • Finns det några rabatter tillgängliga?

    Ja. Vi erbjuder särskilda rabatter för studerande. Registrera dig bara med din mailadress kopplad till skolan så kommer du automatiskt vara kvalificerad för rabatten. Kontakta vårt supportteam ifall din skolmail inte kan identifieras.

  • Kan jag dela fakturering för flera konton?

    Ja. Du kan ha allas konton på ett kreditkort för centraliserad fakturering. Ange bara dina teammedlemmars e-postadresser, så kommer planen automatiskt att uppgraderas.

  • Vilka betalningsmetoder tar ni emot?

    Vi tar emot betalningar via Visa, MasterCard, Discover, American Express med mer. Om du inte vill använda ett kreditkort tar vi emot bland annat PayPal. Om du behöver använda en annan betalningsmetod, hör av dig till oss.

  • Kan jag avsluta min prenumeration?

    Absolut! Om ditt konto inte passar dina behov kan du när som helst avbryta din prenumeration.

  • Vad händer om jag behöver ändra min plan halvvägs genom kontraktet?

    Du kan lägga till och ta bort licenser vid behov från ditt konto. På samma sätt kan du uppgradera din prenumeration, så kommer vi automatiskt att justera priset.

  • Kan ni fakturera mig?

    Ja. Skattefakturor finns tillgängliga för alla planer. Alla priser är inklusive VAT (moms). Vi erbjuder månatlig och årlig fakturering. Du kan hantera dina fakturor och ändra dina faktureringsinställningar från din kontosida.
