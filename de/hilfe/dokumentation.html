Willkommen zu unserem Benutzerleitfaden

Obwohl wir versucht haben, alles besonders einfach zu gestalten, finden Sie hier eine kurze Anleitung, die Ihnen durch den Bearbeitungsvorgang helfen wird.

PDF Organisieren

PDF zusammenfügen

Um zwei oder mehrere PDFs zusammenzufügen, wählen Sie zuerst Ihre Dokumente von Ihrem Gerät oder von IhremGoogle Drive- oder Dropbox-Konto aus. Sie können Ihre Dateien nach Belieben anordnen, bevor Sie sie zu einem Dokument zusammenfügen.

Sie können außerdem mehrere passwortgeschützte Dokumente zu einem Dokument kombinieren. Einfach die gewünschten Dateien hochladen, abwarten, bis die Verarbeitung abgeschlossen ist und anschließend die kombinierte Datei herunterladen. Informieren Sie sich über die "PDFs Kombinieren" Limits für jeden Konto-Typ.

PDF teilen

Möchten Sie ein PDF in mehrere Dateien aufteilen, müssen Sie zuerst das Dokument auf Ihrem Gerät auswählen oder von einem Ihrer Cloud Storage-Konten hochladen, wie z. B. Google Drive oder Dropbox. Sie können außerdem passwortgeschützte Dokumente aufteilen.

Die Datei zum Aufteilen hochladen,. Sie können Ihre Datei entweder nach den verschiedenen Bereichen aufteilen oder alle Seiten extrahieren. Außerdem können Sie alle Bereiche in einem PDF-Dokument zusammenfügen. Klicken Sie einfach auf den Button PDF aufteilen, dann kann es schon losgehen. Informieren Sie sich über die 'PDF Aufteilen' Limits für jeden Abo-Typ.

Seiten entfernen

Möchten Sie Seiten von einem oder mehreren PDFs Entfernen, müssen Sie zuerst Ihre Dateien von Ihrem Gerät oder von einem Ihrer Cloud Storage-Konten hochladen, wie z. B. Google Drive oder Dropbox. Sie können außerdem Seiten von passwortgeschützten Dokumenten entfernen.

Sie werden für jede Seite Ihrer Datei eine Miniaturansicht sehen. Wenn Sie mehr als eine Datei hochladen, werden die Miniaturansichten jeder separaten Datei in unterschiedlichen Farben eingerahmt. Sie können auf jede einzelne Miniaturansicht klicken, um die jeweilige Seite zu entfernen. Nach dem Klick wird über der Miniaturansicht ein rotes Kreuz erscheinen, das anzeigt, dass die Seite entfernt wird. Klicken Sie anschließend auf den Button "Seiten Entfernen", um Ihre bearbeitete PDF-Datei herunterzuladen. Informieren Sie sich über die "Seiten Entfernen"Limits für jeden Konto-Typ.

Seiten extrahieren

Möchten Sie Seiten von einem PDF Extrahieren, müssen Sie zuerst Ihre Dateien von Ihrem Gerät oder von einem Ihrer Cloud Storage-Konten hochladen, wie z. B. Google Drive oder Dropbox. Sie können außerdem Seiten von passwortgeschützten Dokumenten entfernen.

Sie werden für jede Seite Ihrer Datei eine Miniaturansicht sehen. Wenn Sie mehr als eine Datei hochladen, werden die Miniaturansichten jeder separaten Datei in unterschiedlichen Farben eingerahmt. Klicken Sie auf jene Seiten, die Sie extrahieren möchten. Legen Sie fest, ob Sie alle extrahierten Seiten zu einem neuen PDF zusammenfügen möchten oder ob Sie sie in eine .zip-Datei herunterladen möchten. Klicken Sie anschließend auf den Button "Seiten Extrahieren". Informieren Sie sich über die "Seiten Extrahieren"Limits für jeden Konto-Typ.

PDF Organisieren

Möchten Sie die Seiten eines PDFs Organisieren, müssen Sie zuerst Ihre Datei von Ihrem Gerät oder von einem Ihrer Cloud Storage-Konten hochladen, wie z. B. Google Drive oder Dropbox. Sie können außerdem Seiten von passwortgeschützten Dokumenten entfernen.

Sie werden für jede Seite Ihrer Datei eine Miniaturansicht sehen. Wenn Sie mehr als eine Datei hochladen, werden die Miniaturansichten jeder separaten Datei in unterschiedlichen Farben eingerahmt. Anschließend können Sie per Drag & Drop die Miniaturansichten verschieben, um Ihre PDFs ganz nach Belieben neu anzuordnen. Klicken Sie auf den Button "PDF Organisieren", um Ihr neues PDF herunterzuladen. Informieren Sie sich über die "PDF Organisieren" Limits für jeden Konto-Typ.

PDF optimieren

PDF komprimieren

Um PDFs zu komprimieren, wählen Sie die Dokumente von Ihrem Gerät oder von Ihrem Google Drive- oder Dropbox- Konto aus. Anschließend können Sie die Datei oder Dateien Hochladen, um sie zu optimieren. Wählen Sie zwischen einem hohen, mittleren oder niedrigen Komprimierungsgrad aus und klicken Sie anschließend auf den Button "PDF Komprimieren". Je höher der Komprimierungsgrad, desto kleiner ist die Dateigröße und desto niedriger ist die Qualität.

Klicken Sie auf den Button 'PDF Komprimieren', um Ihr verkleinertes PDF herunterzuladen oder speichern Sie es wieder in Ihrer Cloud. Informieren Sie sich über die 'PDF Komprimieren' Limits für jeden Abo-Typ.

PDF bearbeiten

PDF bearbeiten

Um Ihr PDF zu bearbeiten, laden Sie Ihre Datei von Ihrem Gerät oder von Ihrem Google Drive- oder Dropbox-Konto hoch.

Sobald Sie im PDF-Viewer sind, können Sie die Symbolleiste oben benutzen, um durch Ihr Dokument zu navigieren. Klicken Sie auf das Rädchensymbol, um die Einrichtung Ihrer Seite zu bestimmen. Sie können eine einzelne Seite oder mehrere Seiten auf einmal ansehen. Um auf einzelne Seiten und innerhalb der Seiten zu navigieren, wählen Sie das Handsymbol. Nutzen Sie das Vergrößerungslas, um bestimmte Bereiche des Dokuments zu vergrößern bzw. verkleinern.

Um Elemente oder Text zu einer Seite hinzuzufügen, scrollen Sie nach unten zu der Seite, die Sie bearbeiten möchten. Wählen Sie aus den Symbolen für Text, Bild oder Form, die in der Leiste zum Bearbeiten eines PDF am oberen Rand des Viewers verfügbar sind, aus. Nutzen Sie Ihre Maus, um jedes Element beliebig zu skalieren oder auf der gleichen Seite, auf der es erstellt worden ist, zu verschieben.

Um Elemente zu einer anderen Seite hinzuzufügen, navigieren Sie zu der Seite und wählen Sie dann ein neues Element aus der Symbolleiste aus. Sie können die Farbe, Transparenz, Randfarbe, Ausrichtung, das Format oderdie Linienstärke Ihrer Bilder, Formen oder Texte mit Hilfe ihrer Eigenschaftenleisten ändern. Klicken Sie auf die Schaltfläche ' PDF bearbeiten', um Ihre Datei zu bearbeiten.

Seitenzahlen hinzufügen

Um PDF-Seiten zu nummerieren, laden Sie zuerst Ihre Dateien von Ihrem Gerät oder von Ihrem Google Drive- oder Dropbox-Konto hoch. Wählen Sie aus, ob Sie alle Seiten oder nur jede zweite Seite nummerieren möchten. Wählen Sie aus, welche Spanne von Seiten Sie nummerieren möchten. Sie können außerdem in der Toolbar die Position der Seitenzahl und ihr Format (Font, Größe, Farbe, Transparenz etc.) verändern.

Klicken Sie anschließend auf den Button "Seitenzahlen hinzufügen" und laden Sie Ihr nummeriertes Dokument herunter oder speichern Sie es wieder in Ihrer Cloud. Informieren Sie sich über die "Seitenzahlen hinzufügen" Limits für jeden Konto-Typ.

Wasserzeichen hinzufügen

Möchten Sie ein PDF mit einem Wasserzeichen versehen, müssen Sie zuerst die gewünschte PDF-Datei von IhremGerät oder von Ihrem Google Drive oder Dropbox-Konto hochladen. Sie können in Ihrem Wasserzeichen Text und Bild kombinieren. Sämtliche Seiten Ihres Dokuments werden an derselben Stelle mit einem Wasserzeichen versehen. Format, Ausrichtung und Transparenz des Wasserzeichens können Sie frei wählen.

Klicken Sie anschließend auf den Button "Wasserzeichen hinzufügen" und laden Sie Ihr mit einem Wasserzeichen versehenes Dokument herunter oder speichern Sie es wieder in Ihrer Cloud. Informieren Sie sich über die "Wasserzeichen hinzufügen" Limits für jeden Konto-Typ.

PDF reparieren

Möchten Sie ein PDF reparieren, müssen Sie zuerst die beschädigte oder fehlerhafte PDF-Datei von Ihrem Gerät oder von einem Google Drive oder Dropbox-Konto hochladen. Unser Tool wird versuchen, die Inhalte in Ihrer fehlerhaften Datei zu analysieren, zu reparieren und wiederherzustellen.

Klicken Sie anschließend auf den Button "Reparieren" und laden Sie Ihr repariertes Dokument herunter oder speichern Sie es wieder in Ihrer Cloud. Informieren Sie sich über die "Reparieren" Limits für jeden Konto-Typ.

In PDF konvertieren

JPG in PDF

Möchten Sie eine oder mehrere Dateien von JPG zu PDF konvertieren, müssen Sie zuerst Ihre Bilder von Ihrem Gerät oder direkt von Ihrem Google Drive oder Dropbox-Konto hochladen.

Wählen Sie, ob Sie Ihr PDF im Hochformat oder im Querformat formatieren möchten und ob es einen Rand haben soll oder nicht. Sie können außerdem alle einzelnen Bilder in eine einzige PDF-Datei zusammenfügen. Downloaden Sie anschließend Ihr PDF oder speichern Sie es wieder in Ihrer Cloud. Informieren Sie sich über die 'JPG zu PDF' Limits für jeden Abo-Typ.

WORD in PDF

Möchten Sie ein oder mehrere Dokumente von Word zu PDF konvertieren, müssen Sie zuerst die Dokumente von Ihrem Gerät oder von einem Ihrer Cloud Storage-Konten hochladen, wie z. B. Google Drive oder Dropbox.

Klicken Sie auf den Button 'Zu PDF Konvertieren', um Ihre .docx-Datei in ein nicht-editierbares Dokument zu verwandeln. Laden Sie anschließend Ihr PDF herunter oder speichern Sie es wieder in Ihrer Cloud. Informieren Sie sich über die 'Word zu PDF' Limits für jeden Abo-Typ.

POWERPOINT in PDF

Alle Bilder und Texte aus Ihrer .pptx-Präsentation werden in ein PDF-Dokument umgewandelt. Laden Sie anschließend Ihr PDF herunter oder speichern Sie es wieder in Ihrer Cloud. Informieren Sie sich über die 'PowerPoint zu PDF' Limits für jeden Abo-Typ.

Alle Tabellen und Grafiken aus Ihrem Excel-Dokument werden in ein PDF-Dokument umgewandelt. Laden Sie anschließend Ihr PDF herunter oder speichern Sie es wieder in Ihrer Cloud. Informieren Sie sich über die 'Zu PDF Konvertieren' Limits für jeden Abo-Typ.

HTML in PDF

Um eine Webseite in ein PDF zu konvertieren, laden Sie Ihre Bilder von Ihrem Gerät oder direkt von Ihrem Google Drive- oder Dropbox-Konto hoch.

Kopieren Sie die URL, die Sie konvertieren möchten in das Textfeld. Eine Vorschau des Dokuments wird angezeigt werden. Sie können die Ausrichtung und die Ränder des resultierenden Dokuments mit Hilfe des Menüs der Seitenleiste anpassen.

Klicken Sie auf die Schaltfläche 'In PDF konvertieren'. Laden Sie Ihre exportierte Webseite herunter oder speichern Sie sie in der Cloud. Überprüfen Sie die 'HTML in PDF'-Limits für jeden Kontotyp.

PDF unterschreiben

Um Ihr eigenes Dokument zu unterzeichnen

Laden Sie das Dokument in unser Sign PDF-Tool. Unterzeichnen. Wählen Sie Ihre bevorzugte Unterschriftsmethode aus dem Menü der Seitenleiste, um Ihre Unterschrift anzupassen. Sie können Ihre Unterschrift entweder zeichnen, eine bestehende hochladen oder mit Text unterzeichnen. Sie können andere Felder wie Datum und Initialen zu Ihrer Unterschrift hinzufügen. Herunterladen. Klicken Sie auf die Schaltfläche 'PDF herunterladen', um Ihr unterzeichnetes PDF herunterzuladen oder es wieder in der Cloud zu speichern.

Um elektronische Unterschriften zu sammeln

Um eine Unterzeichnungsanfrage zu erstellen, laden Sie das Dokument in unser Sign PDF-Tool hoch. Wenn Sie Ihr PDF-Dokument hochgeladen haben, erstellen Sie eine Unterzeichungsanfrage. Sie werden Unterzeichner hinzufügen, Signaturfelder einfügen, Sicherheitsparameter anpassen und sie dann senden müssen. Unterzeichnen. Alle Unterzeichner werden eine E-Mail mit Zugriff auf das Dokument erhalten und sie werden in der Lage sein, es zu lesen und zu unterzeichnen. Herunterladen. Wenn der Signaturprozess abgeschlossen ist, werden Sie eine Kopie des unterzeichneten Dokuments erhalten zusammen mit der Option, einen Prüfpfad herunterzuladen. Der Prüfpfad kann als Nachweis in Gerichtsverfahren dienen.

Technische Konzepte

Passwort-geschützte Dateien

Wenn Sie eine passwortgeschützte Datei bearbeiten möchten, werden wir Sie bitten, uns beim Öffnen der Datei zu helfen. Nachdem die Datei geöffnet und bearbeitet wurde, verpflichten wir uns, das Passwort nicht auf unseren Servern zu speichern.

Dateien drehen

Sie können Ihre Dateien drehen, indem Sie Ihren Mauszeiger langsam über eine Datei bewegen und das ()-Icon anklicken. Falls Sie mehr als 10 Dateien auf einmal drehen möchten, können Sie das automatisch im Pop-Up-Fenster erledigen, das auftauchen wird.

JPG in PDF Drehung

Beim Drehen eines Bildes ist nur das Bild selbst betroffen. Anschließend haben Sie die Möglichkeit, die Seitenausrichtung (vertikal oder horizental) mit dem gerade gedrehten Bild auszuwählen.

Dateien organisieren

Sie können Ihre Dateien auf unterschiedliche Arten organisieren. Ziehen Sie sie per Drag and Drop an einen beliebigen Ort; oder ordnen Sie sie alphabetisch, indem Sie auf das () Icon klicken; außerdem haben Sie die Möglichkeit, sie in umgekehrter alphabetischer Reihenfolge anzuordnen, indem Sie auf das () Icon klicken.

Dateien komprimieren

Nach der Verarbeitung Ihrer Daten werden Sie die ursprüngliche Dateigröße sehen und um wie viele Prozent sie reduziert wurde. Im Fall von bereits optimierten PDF-Dateien sind wir leider nicht in der Lage, sie weiter zu komprimieren.

Hochladen und verarbeiten

Die Upload- und Bearbeitungsdauer Ihrer Dateien hängt stark von Ihrer Internet-Verbindung und der Größe der gewählten Datei ab. Wenn der Browser während der Verarbeitung Ihrer Datei geschlossen wird, können Sie den Bearbeitungsstatus im Menüpunk "Mein Konto" unter "Zuletzt bearbeitete Dateien" überprüfen.

Wird heruntergeladen...

Die Ausgabedateien werden nach einer gewissen Zeitperiode von unseren Servern gelöscht, abhängig von Ihrem Konto-Typ. Sie können Ihre Dateien auch direkt zurück in die Cloud speichern, z. B. auf IhremGoogle Drive oder Dropbox-Konto. Es steht Ihnen frei, die Gültigkeit Ihres Download-Links jederzeit zu widerrufen.

Von der Cloud aus arbeiten

Die Dateien, die Sie bearbeiten möchten, müssen nicht unbedingt auf Ihrem Gerät gespeichert sein. Wenn Sie ein Google Drive oder Dropbox-Konto haben, können Sie Ihre Dokumente direkt von dort auf unsere Website laden. Sobald die Bearbeitung Ihrer Dateien abgeschlossen ist, können Sie sie direkt zurück in Ihre Cloud speichern. Um in der Cloud gespeicherte Dokumente zu bearbeiten, müssen Sie bei der erstmaligen Nutzung dieser Funktion das Passwort zu Ihrem Cloud-Konto eingeben. Diese Funktion ist besonders nützlich, wenn Sie von einem mobilen Endgerät oder einem Tablet aus arbeiten.

Dateien für den Upload auswählen

Sie können Dateien von Ihrem Computer auswählen, indem Sie auf den Button "Bilder Auswählen" klicken. Wenn Sie mehr als ein Bild auf einmal auswählen möchten, halten Sie die "Ctrl-Taste" gedrückt, während Sie mit der Maus auf weitere Bilder klicken. Falls die Dateien bereits in der richtigen Reihenfolge angeordnet sind, können Sie auf die erste und letzte Datei klicken, während Sie die "Shift-Taste" gedrückt halten. Alternativ können Sie die Dateien per Drag & Drop direkt auf Ihrer Bearbeitungsfläche ablegen.

Mindestanforderungen

Wir empfehlen, dass Sie unsere Tools in einem der folgenden Browser anwenden: Chrome, Firefox, Explorer +10 und Safari. Wenn Sie mit dem Explorer der Versionen 8 und 9 arbeiten, wird die Rendering-Option für Dokument-Seiten deaktiviert. Für die Arbeit mit iLovePDF muss Javascript aktiviert sein.

Technische Probleme

Diese neue Version von iLovePDF wurde designt, um Ihnen die Arbeit zu erleichtern. Außerdem haben wir uns bemüht, etwaige technische Probleme zu lösen. Sollten Sie dennoch auf Schwierigkeiten stoßen, zögern Sie bitte nicht, uns zu informieren. Wenn Ihre Datei keine vertraulichen Inhalte enthält, können Sie sie an Ihre E-Mail an uns anhängen, damit wir das Problem schneller lösen können. Wir werden alles in unserer Macht Stehende unternehmen, um eine Lösung für Ihre Problem zu finden.
