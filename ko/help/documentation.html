PDF 구성

PDF 병합

두 개 이상의 PDF를 병합하려면 기기에서 문서를 선택하거나 Google Drive 또는 Dropbox 계정에서 문서를 선택하십시오. 병합하기 전에 파일을 원하는 대로 정렬할 수 있습니다.

암호로 보호된 여러 문서를 하나로 결합할 수도 있습니다. 선택한 파일을 업로드하여 처리 한 다음, 병합된 파일을 다운로드 하십시오. 계정 유형별 'PDF 병합' 한도를 확인하십시오.

PDF 분리

PDF를 여러 파일로 분할하려면 기기에서 문서를 선택하거나 Google Drive 또는 Dropbox 같은 클라우드 스토리지 계정에서 파일을 선택하십시오. 암호로 보호된 파일을 분할할 수도 있습니다.

분할할 파일을 업로드하십시오. 범위 별로 파일을 분할하거나 모든 페이지를 추출할 수 있습니다. 하나의 PDF 파일에서 모든 범위를 병합할 수도 있습니다. PDF 분할 버튼을 클릭하기만 하면 됩니다. 계정 유형별 'PDF 분할' 한도를 확인하십시오.

페이지 제거

하나 이상의 PDF에서 페이지를 제거하려면 기기에서 파일을 업로드 하거나 Google Drive 또는 Dropbox 같은 클라우드 연결 계정에서 파일을 다운로드 하십시오. 암호로 보호된 문서의 페이지도 삭제할 수 있습니다.

모든 페이지의 미리보기 이미지가 표시됩니다. 여러 파일을 업로드 할 때 각 파일의 미리보기는 다른 색상으로 표시됩니다. 각 미리보기를 클릭하여 해당 페이지를 제거할 수 있습니다. 페이지를 제거하면 미리보기 이미지 위에 빨간색 십자가 표시가 나타납니다. 페이지 제거 버튼을 누르면 처리된 PDF를 다운로드 할 수 있습니다. 계정 유형별 '페이지 제거' 한도를 확인하십시오.

페이지 추출

PDF 페이지를 추출하려면 기기에서 파일을 업로드 하거나 Google Drive 또는 Dropbox 같은 클라우드 스토리지에서 업로드 하십시오. 암호로 보호된 문서의 페이지도 삭제할 수 있습니다.

모든 페이지의 미리보기 이미지가 표시됩니다. 두 개 이상의 파일을 업로드 할 때 각 파일의 미리보기 이미지는 다른 색상으로 표시됩니다. 추출할 페이지를 클릭하십시오. 추출된 모든 페이지를 새 PDF로 병합할지, .zip 폴더로 다운로드할지 선택한 후, 페이지 추출 버튼을 누르십시오. 계정 유형별 '페이지 추출' 한도를 확인하십시오.

PDF 구성

모든 페이지의 미리보기 이미지가 표시됩니다. 두 개 이상의 파일을 업로드 할 때 각 파일의 미리보기 이미지는 다른 색상으로 표시됩니다. 미리보기 이미지를 제거하거나 끌어다 놓으며 원하는 대로 PDF 페이지를 재정렬할 수 있습니다. 'PDF 구성' 버튼을 클릭하여 새 PDF 파일을 다운로드하십시오. 계정 유형별 'PDF 구성' 한도를 확인하십시오.

PDF 최적화

PDF 압축

PDF를 압축하려면 기기에서 문서를 선택하거나 Google Drive 또는 Dropbox 계정에서 문서를 선택하십시오. 최적화 할 파일을 업로드하십시오. 압축 수준을 최대, 높음, 일반 중에서 선택하고 압축 PDF 버튼을 누르십시오. 압축률이 높을수록 파일 크기가 작아지고 품질이 낮아집니다.

압축된 PDF를 다운로드하거나 클라우드에 다시 저장하려면 'PDF 압축' 버튼을 클릭하십시오. 계정 유형별 'PDF 압축' 한도를 확인하십시오.

PDF 편집

PDF 편집

PDF를 편집 하려면 기기 , 구글 드라이브 또는 드롭박스에서 파일을 업로드하십시오.

PDF 뷰어 내부 상단 툴바를 사용하여 문서를 탐색 합니다. 바퀴 아이콘을 클릭하여 페이지를 설정하세요. 한 번에 한 페이지 또는 여러 페이지를 볼 수 있습니다. 개별 페이지로 이동하려면 손 모양 아이콘을 선택하십시오. 돋보기를 사용하여 문서의 특정 부분을 확대 및 축소할 수 있습니다.

요소를 추가 하거나 페이지의 텍스트를 편집하려면 편집하려는 페이지 아래로 스크롤하세요. 뷰어 상단의 PDF 편집 도구 모음에서 사용 가능한 텍스트 , 이미지 또는 도형 아이콘 중에서 선택합니다. 마우스를 사용해 원하는 요소를 크기 조정 또는 이동 합니다.

다른 페이지에 요소를 추가하려면, 페이지로 이동한 다음 도구바에서 새 항목을 선택하십시오. 속성 표시 줄을 사용하여 이미지, 도형 또는 텍스트의 색상, 투명도, 테두리 색상, 회전, 형식 또는 선 굵기 를 변경합니다. 'PDF 편집' 버튼을 클릭하여 파일을 처리하십시오.

페이지 번호 추가

PDF에 페이지 번호를 매기려면 기기에서 파일을 업로드 하거나 Google Drive 또는 Dropbox 계정에서 파일을 업로드 하십시오. 모든 페이지에 번호를 매길지, 마주보는 페이지에만 번호를 매길지 선택하십시오. 번호를 매길 페이지 범위를 선택하십시오. 툴바에서 번호의 위치와 형식(폰트, 크기, 색상, 불투명도...)을 수정할 수 있습니다.

'페이지 번호 추가' 버튼을 클릭하고 번호가 매겨진 PDF를 다운로드하거나 클라우드에 다시 저장하십시오. 계정 유형별 '페이지 번호' 한도를 확인하십시오.

'워터마크 추가' 버튼을 클릭하고 워터마크가 삽입된 PDF를 다운로드하거나 클라우드에 다시 저장하십시오. 계정 유형별 '워터마크' 한도를 확인하십시오.

PDF 해제

PDF의 암호를 제거하려면 기기에서 문서를 업로드 하거나 Google Drive 또는 Dropbox 계정에서 직접 업로드 하십시오.

'PDF 잠금 해제' 버튼을 클릭하십시오. 도구는 암호를 제거하여 문서를 잠금 해제합니다. 계정 유형별 'PDF 잠금해제' 한도를 확인하십시오.

PDF 보호

PDF를 암호화하여 보호하려면 기기에서 문서를 업로드 하거나 Google Drive 또는 Dropbox 계정에서 직접 파일을 업로드 하십시오.

PDF를 암호화하고 보호할 사용자 암호를 선택하십시오. 그런 다음 'PDF 보호' 버튼을 클릭하여 파일에 암호를 추가하십시오. 계정 유형별 'PDF 보호' 한도를 확인하십시오.

PDF PDF/A로 변환

PDF를 PDF/A로 변환하거나 일괄 변환하려면 기기에서 문서를 업로드 하거나 Google Drive 또는 Dropbox 계정에서 직접 업로드 하십시오.

'PDF를 PDF/A로 변환' 버튼을 클릭하고 변환된 PDF를 다운로드하거나 클라우드에 다시 저장하여 장기간 보관하십시오. 계정 유형에 따라 'PDF를 PDF/A로 변환'한도를 확인하십시오.

PDF에 서명하기

자신의 문서에 서명하려면

PDF 서명 도구에 파일을 업로드합니다. 서명 . 사이드 바 메뉴에서 원하는 서명 스타일을 선택하여 나만의 서명을 이용할 수 있습니다. 서명을 그리거나 기존 서명을 업로드 또는 텍스트를 서명으로 사용할 수 있습니다. 날짜와 이니셜과 같은 다른 필드도 추가할 수 있습니다. 다운로드. "PDF 서명" 버튼을 클릭하여 서명된 PDF 파일을 다운로드하거나 클라우드 스토리지에 다시 저장합니다.

전자 서명을 수집하려면

서명 요청을 하려면 서명 PDF 도구에 문서를 업로드하십시오. PDF 문서를 업로드한 후 서명 요청을 작성 합니다. 서명자를 추가하고 서명 필드를 포함하며 보안 매개 변수를 조정한 후 전송해야합니다. 서명. 모든 서명자는 문서에 접근할 수 있는 이메일을 받으며, 이를 읽고 서명 할 수 있습니다. 다운로드 . 서명 프로세스가 완료되면 서명된 문서의 사본과 감사 내역 다운로드 옵션이 제공됩니다. 감사 추적은 법적 절차에서 증거로 사용될 수 있습니다.

기술 개념

암호로 보호된 파일

암호로 보호된 파일을 처리하는 과정에서 파일을 열어달라는 요청을 할 것입니다. 일단 파일을 열고 처리한 후에는 서버에 암호를 저장하지 않습니다.

파일 회전

마우스를 파일 위에 올려 놓고 () 아이콘을 클릭하여 파일을 회전시킬 수 있습니다. 그러나 한 번에 10개 이상의 파일을 회전하려면 팝업 창에서 자동으로 회전할 수 있습니다. 그 전에 모든 페이지를 같은 방향으로 회전시켜야 합니다.

JPG에서 PDF 회전

PDF 문서 이미지를 회전하면 이미지에만 영향을 줍니다. 새로 회전된 이미지로 페이지 방향(세로 또는 가로)을 선택할 수 있습니다.

파일 구성

파일은 여러 가지 방법으로 구성할 수 있습니다. 원하는 대로 끌어다 놓으십시오. () 아이콘을 클릭하여 알파벳순으로 정렬할 수도 있습니다. () 아이콘을 클릭하면 알파벳 역순으로 정렬할 수 있습니다.

파일 압축

파일을 처리 한 후 원래 파일 크기와 줄어든 비율이 표시됩니다. PDF 파일이 이미 최적화 되었다면 더 이상 압축할 수 없습니다.

업로드 및 처리

파일 업로드와 처리 시간은 인터넷 연결과 선택한 파일 크기에 따라 다릅니다. 파일을 처리하는 동안 브라우저가 닫히면 '내 계정' 메뉴의 '최근 처리된 파일' 섹션에서 처리 상태를 확인할 수 있습니다.

다운로드중

출력 파일은 사용자 유형에 따라 일정 기간 후에 서버에서 삭제됩니다. Google Drive 또는 Dropbox 계정에서 클라우드로 다시 저장할 수 있습니다. 언제든지 다운로드 링크의 유효 기간을 취소할 수 있습니다.

클라우드에서 가져오기

반드시 기기에 저장된 파일만 처리할 수 있는 건 아닙니다. Google Drive 또는 Dropbox 계정이 있다면 직접 파일을 우리 페이지로 다운로드 할 수 있습니다. 파일이 처리되면 다시 클라우드에 저장할 수 있습니다. 이 기능을 처음 사용할 때 클라우드 계정의 암호를 입력하면 클라우드에 저장된 파일을 처리할 수 있습니다. 이 기능은 모바일 기기 또는 태블릿을 사용할 때 특히 유용합니다.

업로드 할 파일 선택

컴퓨터에서 파일을 선택하려면 '이미지 선택'' 버튼을 클릭하십시오. 여러 파일을 동시에 선택하려면 'Ctrl' 키를 누른 상태에서 파일을 클릭하십시오. 파일이 연속된 순서로 있는 경우 'Shift' 키를 누른 상태에서 처음과 마지막을 클릭하십시오. 파일을 캔버스로 직접 끌어다 놓을 수도 있습니다.

최소 요구 사항

Chrome, Firefox, Explorer +10 및 Safari와 같은 브라우저에서 도구를 사용하는 것이 좋습니다. Explorer 버전 8과 9에서 작업할 땐 문서 페이지의 렌더링 옵션이 비활성화됩니다. iLovePDF는 Javascript를 활성화한 상태에서 작업해야 합니다.
