Tim dukungan kami menjawab pertanyaan-pertanyaan berikut hampir setiap hari

Kami pikir peralatan itu pun bisa berguna bagi Anda

  • Apakah Anda menyimpan salinan file saya yang diproses?

    Sama sekali tidak. File Anda mutlak milik Anda. Walaupun file tersebut berada di server kami, file itu sangat aman dan tidak ada seorang pun yang dapat mengaksesnya. Kami hanya menyimpannya maksimal selama 2 jam, agar Anda dapat mengunggahnya. Setelahnya, file itu dihapus dari server kami. Anda juga dapat menghapus sendiri dokumen tersebut di akhir konversi. Kami tidak akan memeriksa, menyalin atau menganalisis file Anda dengan cara apa pun.

  • Apakah arsip perusahaan aman dengan layanan Anda?

    Ya. Semua unggahan menggunakan https/SSL dan dilengkapi enkripsi end-to-end kami agar privasi lebih terjaga. Langkah-langkah tambahan ini meningkatkan keamanan dan memenuhi sebagian besar kebijakan privasi data perusahaan. Sederhananya, data Anda diproses seaman mungkin. Kami mematuhi peraturan Perlindungan Data Uni Eropa, yang merupakan salah satu standar keselamatan paling ketat di dunia (GDPR).

  • Apa saja persyaratan sistem Anda?

    Kami memiliki persyaratan sistem yang sangat mendasar. Agar alat kami dapat berjalan lancar, kami menyarankan agar Anda bekerja dengan peramban-peramban web berikut ini: Chrome, Firefox, Explorer +10 dan Safari. Jika suatu saat Anda menemukan masalah dalam layar unduhan, kami menyarankan agar Anda mengatur peramban Anda untuk berjalan di mode Penyamaran.

  • Mengapa saya tidak menerima konfirmasi alamat email saya?

    Email konfirmasi akan segera terkirim ke akun email Anda. Meskipun demikian, kadang-kadang memakan waktu cukup lama. Jika setelah 30 menit tidak ada yang masuk ke kotak masuk, periksa Folder Spam/Trash atau hubungi kami via email.

  • Bagaimana cara mengunggah file?

    Cara termudah adalah dengan menyeret dan menjatuhkan file Anda langsung ke kanvas kami. Anda juga dapat mengklik tombol 'Pilih file' atau mengimpor file dari sistem penyimpanan cloud utama seperti Google Drive atau Dropbox.

  • Bisakah saya mengkonversi PDF hasil pemindaian menjadi dokumen yang dapat diedit?

    Untuk mengonversi PDF hasil pemindaian menjadi PDF yang dapat diedit atau format lain yang dapat diedit, diperlukan adanya OCR (Optical Character Recognition), yakni sistem yang mengonversi teks yang tidak dapat dipilih dan hasil pemindaian menjadi dokumen Office. Sejak rilis terakhir kami, dimungkinkan untuk melakukan konversi PDF Hasil Pemindaian ke Office.

  • Mengapa konversi saya memakan waktu yang sangat lama?

    Sayangnya, hal ini bukan semata-mata tergantung kami. Kecepatan pemrosesan tergantung pada banyak faktor, jadi waktu yang akan Anda perlukan untuk mengonversi file akan tergantung juga pada kecepatan koneksi Internet Anda sendiri, ukuran file-file terpilih dan, terutama, pada seberapa sibuknya server kami.

  • Bisakah saya bekerja dari cloud?

    Ya! File yang ingin Anda proses bahkan tidak perlu berada di perangkat tempat Anda bekerja. Jika Anda memiliki Akun Google Drive atau Dropbox, Anda dapat mengimpor file langsung ke halaman kami, memprosesnya, lalu menyimpannya kembali ke cloud. Sebenarnya, ini metode yang paling tepat ketika menggunakan iLovePDF App dari ponsel atau tablet Anda karena membantu Anda mengurangi penggunaan mobile data.
