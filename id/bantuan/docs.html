Selamat datang di panduan pengguna kami

Meskipun kami telah mencoba menyederhanakannya, berikut ini panduan singkat untuk membantu Anda melakukan proses pengeditan.

Atur PDF

Gabungkan PDF

Untuk menggabungkan dua PDF atau lebih, pilih dokumen dari perangkat atau dari akun Google Drive atau Dropbox Anda. Anda dapat mengatur file sesuka hati sebelum menggabungkannya.

Anda juga dapat menggabungkan beberapa dokumen yang diproteksi dengan kata sandi. Unggah file terpilih, proses, lalu unduh file hasil penggabungan. Cari tahu batas 'Gabung PDF' untuk setiap jenis akun.

Pisahkan PDF

Untuk memisahkan PDF menjadi beberapa file, unggah dokumen dari perangkat Anda atau dari akun penyimpanan cloud, misalnya Google Drive atau Dropbox. Anda juga dapat membagi dokumen yang terproteksi kata sandi.

Unggah file yang akan dipisahkan. Anda bisa memisahkan file berdasarkan rentang atau mengekstrak semua halaman. Anda juga dapat menggabungkan semua rentang menjadi satu file PDF. Tekan tombol Pisahkan PDF, maka selesai. Cari tahu batas 'Pisahkan PDF' untuk setiap jenis akun.

Hapus Halaman

Untuk menghapus halaman dari satu atau beberapa PDF, unggah file dari perangkat Anda atau unduh langsung dari akun yang terhubung dengan cloud, misalnya Google Drive atau Dropbox. Anda juga dapat menghapus halaman dari dokumen yang terproteksi kata sandi.

Anda akan melihat gambar mini dari setiap halaman di file. Ketika mengunggah beberapa file, gambar mini setiap file akan dibingkai dengan warna yang berlainan. Anda dapat mengklik setiap gambar mini untuk menghapus halaman terkait. Setelah mengkliknya, akan muncul tanda silang merah di gambar mini itu, yang menunjukkan bahwa halaman tersebut akan dihapus. Klik tombol 'Hapus Halaman' untuk mengunduh PDF yang sudah diproses. Cari tahu batas 'Hapus Halaman' untuk setiap jenis akun.

Ekstrak Halaman

Untuk mengekstrak halaman dari file PDF, unggah file dari perangkat Anda atau dari akun penyimpanan cloud, misalnya Google Drive atau Dropbox. Anda juga dapat menghapus halaman dari dokumen yang terproteksi kata sandi.

Anda akan melihat gambar mini dari setiap halaman di file. Ketika mengunggah lebih dari satu file, gambar mini setiap file akan dibingkai dengan warna yang berlainan. Klik pada halaman yang ingin Anda ekstrak. Tentukan apakah Anda ingin menggabungkan semua halaman yang diekstrak menjadi PDF baru atau mengunduhnya ke folder .zip. Kemudian, klik tombol 'Ekstrak Halaman'. Cari tahu batas 'Ekstrak halaman' untuk setiap jenis akun.

Atur PDF

Untuk mengatur halaman dari file PDF, unggah file dari perangkat Anda atau unduh langsung dari cloud melalui akun Google Drive atau Dropbox. Anda juga dapat menghapus halaman dari dokumen yang terproteksi kata sandi.

Anda akan melihat gambar mini dari setiap halaman di file. Ketika mengunggah lebih dari satu file, gambar mini setiap file akan dibingkai dengan warna yang berlainan. Selanjutnya Anda dapat menghapus atau menyeret dan menjatuhkan gambar mini tersebut untuk mengatur ulang halaman PDF sesuai keinginan. Klik tombol 'Atur PDF' untuk mengunduh PDF yang baru. Cari tahu batas 'Atur PDF' untuk setiap jenis akun.

Optimalkan PDF

Kompres PDF

Untuk mengompres file PDF, pilih dokumen dari perangkat atau dari akun Google Drive atau Dropbox Anda. Unggah file untuk dioptimalkan. Pilih tingkat kompresi ekstrem, tinggi atau biasa, lalu klik tombol Kompres PDF. Semakin tinggi kompresinya, semakin kecil ukuran file dan semakin rendah kualitasnya.

Klik tombol 'Kompres PDF' untuk mengunduh PDF yang sudah diperkecil ukurannya atau menyimpannya kembali di cloud. Cari tahu batas 'Kompres PDF' untuk setiap jenis akun.

Edit PDF

Edit PDF

Untuk mengedit PDF, unggah file dari perangkat Anda atau dari akun Google Drive atau Dropbox Anda.

Setelah masuk ke PDF viewer, gunakan toolbar atas untuk menelusuri dokumen. Klik ikon roda untuk menentukan pengaturan halaman. Anda bisa melihat satu halaman atau beberapa halaman sekaligus. Untuk menelusuri setiap halaman, pilih ikon tangan. Gunakan kaca pembesar untuk memperbesar dan memperkecil bagian-bagian tertentu dari dokumen.

Untuk menambahkan elemen atau teks ke sebuah halaman, gulir ke bawah sampai ke halaman yang ingin Anda edit. Pilih ikon Teks, Gambar atau Bentuk yang tersedia di toolbar Edit PDF di atas viewer. Gunakan mouse untuk mengubah ukuran atau memindahkan elemen yang Anda inginkan di halaman tempat dibuatnya.

Untuk menambahkan elemen ke halaman lain, masuk ke halaman tersebut, lalu pilih item baru dari toolbar. Ubah warna, transparansi, warna border, rotasi, format atau ketebalan garis gambar, bentuk, atau teks dengan menggunakan bilah properti. Klik tombol ‘Edit PDF’ untuk memproses file Anda.

Tambah nomor halaman

Untuk menomori halaman PDF, unggah file dari perangkat atau dari akun Google Drive atau Dropbox Anda. Tentukan apakah Anda ingin menomori semua halaman atau hanya halaman yang berhadapan. Pilih rentang halaman untuk dinomori. Anda juga dapat mengedit posisi nomor, dan formatnya (jenis, ukuran, warna, kegelapan…) pada bilah alat.

Selanjutnya, klik tombol 'Tambah Nomor Halaman' dan unduh PDF yang telah bernomor halaman atau simpan kembali ke cloud. Cari tahu batas 'Nomor Halaman' untuk setiap jenis akun.

Tambahkan tanda air

Untuk memberi tanda air di file PDF, unggah PDF dari perangkat atau dari akun Google Drive atau Dropbox. Anda dapat menggabungkan teks dan gambar di tanda air itu, yang akan ditempelkan pada posisi yang sama di semua halaman. Anda bisa mengatur format, orientasi dan kegelapan khusus.

Selanjutnya, klik tombol 'Tambah Tanda Air' dan unduh PDF yang telah bertanda air atau simpan kembali ke cloud. Cari tahu batas 'Tanda Air' untuk setiap jenis akun.

Perbaiki PDF

Untuk memperbaiki PDF, unggah file PDF Anda yang rusak dari perangkat atau dari akun Google Drive atau Dropbox. Alat kami akan mencoba menganalisis, memperbaiki atau memulihkan isi dari file Anda yang rusak.

Selanjutnya, klik tombol 'Perbaiki' dan unduh PDF yang telah diperbaiki atau simpan kembali ke cloud. Cari tahu batas 'Perbaiki PDF' untuk setiap jenis akun.

Konversi ke PDF

JPG ke PDF

Untuk mengonversi satu atau beberapa file dari JPG ke PDF, unggah gambar dari perangkat atau langsung dari akun Google Drive atau Dropbox Anda.

Pilih apakah Anda ingin PDF keluaran dalam modus potret atau lanskap dan disertai margin atau tidak. Anda juga dapat menggabungkan semua gambar dalam sebuah file PDF. Kemudian, unduh PDF itu atau simpan kembali ke cloud. Cari tahu batas 'JPG ke PDF' untuk setiap jenis akun.

WORD ke PDF

Untuk mengonversi satu dokumen atau lebih dari WORD ke PDF, pilih dokumen dari perangkat Anda atau dari akun penyimpanan cloud Anda seperti akun Google Drive atau Dropbox.

Klik 'Konversi ke PDF' untuk mengubah file .docx menjadi dokumen yang tidak dapat diedit. Selanjutnya, unduh PDF yang dihasilkan atau simpan kembali ke cloud. Cari tahu batas 'Word ke PDF' untuk setiap jenis akun.

POWERPOINT ke PDF

Untuk mengonversi satu atau beberapa file dari Power Point ke PDF, unggah dokumen dari perangkat Anda atau langsung dari Cloud melalui Google Drive atau Dropbox.

Klik 'Konversi ke PDF' untuk mengubah file .pptx menjadi dokumen yang tidak dapat diedit. Semua gambar dan teks dari presentasi .pptx itu akan dikonversi menjadi dokumen PDF. Selanjutnya, unduh file PDF yang dihasilkan atau simpan kembali ke cloud. Cari tahu batas 'Power Point ke PDF' untuk setiap jenis akun.

EXCEL ke PDF

Untuk mengonversi satu atau beberapa file dari Excel ke PDF, unggah dokumen dari perangkat Anda atau langsung dari akun Google Drive atau Dropbox.

Klik 'Konversi ke PDF' untuk mengubah file .xlsx menjadi dokumen yang tidak dapat diedit. Semua tabel dan grafik dari Excel itu akan dikonversi menjadi dokumen PDF. Selanjutnya, unduh file PDF yang dihasilkan atau simpan kembali ke cloud. Cari tahu batas 'Excel ke PDF' untuk setiap jenis akun.

HTML ke PDF

Untuk ubah halaman web ke PDF, unggah gambar dari perangkat Anda atau langsung dari akun Google Drive atau Dropbox Anda.

Tempelkan URL yang ingin Anda konversi di kotak teks. Selanjutnya, akan ditampilkan pratinjau dokumen. Melalui menu sidebar, Anda bisa menyesuaikan orientasi dan margin dokumen yang dihasilkan.

Klik tombol 'Konversi ke PDF'. Unduh halaman web yang diekspor atau simpan lagi di Cloud. Periksa batas 'HTML ke PDF' untuk setiap jenis akun.

Konversi dari PDF

PDF ke JPG

Untuk mengonversi PDF ke gambar JPG, pilih file dari perangkat Anda atau dari akun Google Drive atau Dropbox.

Anda dapat mengubah PDF menjadi JPG atau mengekstrak semua gambar di file .zip. Selanjutnya, klik tombol 'Konversi ke JPG' untuk mengubah PDF menjadi JPG. Cari tahu batas 'PDF ke JPG' untuk setiap jenis akun.

PDF ke WORD

Untuk mengonversi satu dokumen atau lebih dari PDF ke Word, pilih dokumen dari perangkat Anda atau dari akun penyimpanan cloud Anda seperti akun Google Drive atau Dropbox.

Klik 'Konversi ke Word' untuk mengubah PDF menjadi dokumen yang dapat diedit. Selanjutnya, unduh file .docx yang dihasilkan atau simpan kembali ke cloud. Cari tahu batas 'PDF ke Word' untuk setiap jenis akun.

PDF ke POWERPOINT

Untuk mengonversi file PDF ke Power Point, unggah dokumen dari perangkat Anda atau langsung dari cloud melalui Google Drive atau Dropbox.

Klik 'Konversi ke Power Point' untuk mengubah PDF menjadi dokumen yang dapat diedit. Semua gambar, teks dan grafik dari PDF itu akan dikonversi menjadi dokumen Power Point. Ketika mengonversi dokumen yang berisi banyak halaman, setiap halaman akan diubah menjadi slide .pptx. Selanjutnya, unduh file .pptx yang dihasilkan atau simpan kembali ke cloud. Cari tahu batas 'PDF ke Power Point' untuk setiap jenis akun.

PDF ke EXCEL

Untuk mengonversi satu atau beberapa file dari PDF ke Excel, unggah dokumen dari perangkat Anda atau langsung dari akun Google Drive atau Dropbox.

Klik 'Konversi ke Excel' untuk mengubah PDF menjadi dokumen yang dapat diedit. Semua tabel dan grafik dari PDF itu akan dikonversi menjadi dokumen .xslx. Selanjutnya, unduh file Excel yang dihasilkan atau simpan kembali ke cloud. Cari tahu batas 'PDF ke Excel' untuk setiap jenis akun.

Keamanan PDF

Buka PDF Terkunci

Untuk menghapus kata sandi dari file PDF, unggah dokumen dari perangkat atau langsung dari akun Google Drive atau Dropbox Anda.

Selanjutnya klik tombol 'Buka PDF'. Alat kami akan membuka kunci dokumen Anda dengan menghapus kata sandinya. Cari tahu batas 'Aktifkan PDF' untuk setiap jenis akun.

Proteksi PDF

Untuk mengamankan PDF dengan kata sandi, unggah dokumen dari perangkat atau langsung dari akun Google Drive atau Dropbox Anda.

Pilih kata sandi pengguna untuk mengenkripsi dan mengamankan PDF. Kemudian, klik tombol 'Proteksi PDF' untuk menambahkan kata sandi ke file Anda. Cari tahu batas 'Proteksi PDF' untuk setiap jenis akun.

PDF ke PDF/A

Untuk mengonversi PDF ke PDF/A satu per satu atau sekaligus, unggah dokumen dari perangkat Anda atau langsung dari akun Google Drive atau Dropbox.

Selanjutnya, klik tombol 'PDF ke PDF/A' dan unduh PDF yang telah dikonversi atau simpan kembali ke cloud untuk diarsipkan jangka panjang. Cari tahu batas 'PDF ke PDF A' untuk setiap jenis akun.

Tanda tangani PDF

Untuk menandatangani dokumen Anda sendiri

Unggah dokumen ke alat Sign PDF kami. Tanda Tangani. Pilih tanda tangan yang Anda sukai dari menu sidebar untuk menyesuaikannya. Anda bisa menggambar tanda tangan, mengunggah yang sudah ada atau menandatangani dengan teks. Anda juga bisa menambahkan kotak isian lain ke tanda tangan Anda seperti tanggal dan inisial. Unduh. Klik tombol 'Tanda Tangani PDF' untuk mengunduh PDF yang sudah ditandatangani atau menyimpannya kembali ke Cloud.

Untuk mengumpulkan e-Signature

Untuk membuat permintaan tanda tangan, unggah dokumen ke alat Sign PDF kami. Setelah Anda mengunggah dokumen PDF, buat permintaan tanda tangan. Anda perlu menambahkan penandatangan, menyertakan bagian penandatanganan, menyesuaikan parameter keamanan, lalu mengirimkannya. Tanda Tangani. Semua penandatangan akan menerima email yang berisi akses ke dokumen, dan mereka akan bisa membaca serta menandatangani dokumen tersebut. Unduh. Setelah proses tanda tangan selesai, Anda akan menerima salinan dokumen yang ditandatangani, berikut opsi untuk mengunduh Jejak Audit. Jejak Audit bisa digunakan sebagai bukti pada proses hukum.

Konsep teknis

File yang dilindungi kata sandi

Jika Anda memproses file yang diproteksi kata sandi, kami akan meminta Anda untuk membantu kami membukanya. Setelah file itu dibuka dan diproses, kami berkomitmen tidak akan menyimpan kata sandi tersebut di server kami.

Memutar file

Anda dapat memutar file dengan mengarahkan kursor ke file, lalu mengklik ikon (). Tetapi jika Anda ingin memutar lebih dari 10 file sekaligus, Anda dapat melakukannya secara otomatis di jendela pop-up yang akan muncul. Sebelum melakukannya, Anda harus me-render semua halaman agar dapat diputar dengan arah yang sama.

Rotasi JPG ke PDF

Memutar gambar dalam dokumen PDF hanya berpengaruh terhadap gambarnya. Setelah itu, Anda akan dapat memilih orientasi halaman (vertikal atau horizontal) dengan gambar yang baru diputar.

Mengatur file

File dapat diatur dengan berbagai cara. Seret dan jatuhkan sekehendak hati; Anda pun dapat mengaturnya secara alfabetis dengan mengklik ikon (); dan Anda pun memiliki pilihan untuk mengaturnya dengan urutan abjad terbalik dengan mengklik ikon ().

Mengompres file

Setelah file Anda diproses, Anda akan melihat ukuran file asli dan berapa persentase pengurangannya. Untuk file PDF yang sudah dioptimalkan, kami tidak akan dapat mengompresnya lebih lanjut.

Mengunggah dan memproses

Waktu untuk memproses unggahan dan unduhan file akan tergantung pada koneksi Internet Anda dan ukuran file yang telah Anda pilih. Jika peramban web ditutup saat file Anda sedang diproses, Anda dapat memverifikasi status pemrosesannya di menu 'Akun Saya' pada bagian 'File yang terakhir diproses'.

Mengunduh

File keluaran akan dihapus dari server kami setelah waktu tertentu, tergantung pada jenis pengguna Anda. Anda pun dapat menyimpannya langsung ke cloud, di akun Google Drive atau Dropbox. Anda dapat membatalkan validitas tautan unduhan kapan pun Anda mau.

Bekerja dari cloud

Anda tidak perlu menyimpan file yang ingin Anda proses di perangkat Anda. Jika Anda memiliki Akun Google Drive atau Dropbox, Anda dapat mengunduhnya langsung ke halaman kami. Setelah file Anda diproses, Anda bisa menyimpannya kembali ke cloud. Untuk memproses file yang disimpan di cloud, Anda perlu memasukkan kata sandi akun cloud Anda saat pertama kali Anda menggunakan fitur ini. Kata sandi tidak akan disimpan di sistem kami. Fitur ini berguna ketika bekerja dari perangkat seluler atau tablet.

Pilih file yang akan diunggah

Untuk memilih file dari komputer, Anda bisa melakukannya dengan menekan tombol 'Pilih gambar' Jika Anda ingin memilih lebih dari satu file secara bersamaan, tahan tombol 'Ctrl' sambil mengklik file-file tersebut. Jika file-filenya berurutan, klik pada yang pertama dan terakhir, sambil menahan tombol 'Shift'. Anda juga dapat menyeret dan menjatuhkan file-file itu langsung ke kanvas kami.

Persyaratan minimal

Kami menyarankan untuk menggunakan peralatan kami dari peramban-peramban web berikut ini: Chrome, Firefox, Explorer +10 dan Safari. Ketika bekerja dari Explorer versi 8 dan 9, opsi rendering untuk halaman dokumen akan dinonaktifkan. iLovePDF perlu berfungsi dengan javascript diaktifkan.

Masalah teknis

Versi iLovePDF yang baru ini sudah dirancang agar jauh lebih mudah digunakan, dan kami telah mencoba mengatasi segala gangguan teknis. Namun, jika Anda menemukan masalah, jangan ragu untuk menghubungi kami. Jika suatu file tidak mengandung materi rahasia, lampirkan ke email Anda agar kami dapat lebih cepat menyelesaikan masalah. Kami akan berbuat maksimal untuk menghasilkan solusi.
