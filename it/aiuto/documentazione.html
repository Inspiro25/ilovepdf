Benvenuto sulla nostra guida dell'utente

Anche se abbiamo cercato di renderlo davvero semplice, ecco una breve guida per aiutarti nel processo di modifica.

Organizza PDF

Unire PDF

Per unire due o più PDF, seleziona i documenti dal tuo dispositivo o dal tuo account di Google Drive o Dropbox. Puoi organizzare i file come preferisci prima di unirli.

Puoi anche combinare molteplici documenti protetti da password in uno solo. Caricai file selezionati, elaborali, e poi scarica il tuo file unito. Controlla i limiti di 'Unisci PDF' per ogni tipo di account.

Dividi PDF

Per dividere un PDF in diversi file, seleziona il documento dal tuo dispositivo o dal tuo account del cloud di archiviazione come Google Drive o Dropbox. Puoi anche dividere documenti protetti da password.

Carica il file da dividere. È possibile dividere il file per intervalli o estrarre tutte le pagine. È anche possibile unire tutte gli intervalli in un unico file PDF. Premi il tasto Dividi PDF e sarai pronto per iniziare. Controlla i limiti di 'Dividi PDF' per ogni tipo di account.

Rimuovi Pagine

Per rimuovere pagine da uno o più PDF, carica il tuo file dal tuo dispositivo o scaricalo dal tuo account connesso al cloud di Google Drive o Dropbox. Puoi anche rimuovere pagine dai tuoi documenti protetti da password.

Vedrai una miniatura di ogni pagina del tuo file. Quando si caricano più file, le miniature di ciascun file verranno incorniciate con un colore diverso. Puoi fare clic su ciascuna miniatura per rimuovere quella pagina. Dopo aver cliccato, una croce rossa apparirà sulla miniatura, mostrando che la pagina verrà rimossa. Clicca sul pulsante Rimuovi Pagine per scaricare il tuo PDF elaborato. Controlla i limiti di 'Rimuovi Pagine' per ogni tipo di account.

Estrai Pagine

Per estrarre pagine da un PDF, carica il tuo file dal tuo dispositivo o dal tuo account di archiviazione come Google Drive o Dropbox. Puoi anche rimuovere pagine dai documenti protetti da password.

Vedrai una miniatura di ogni pagina del tuo file. Quando si caricano più file, le miniature di ciascun file saranno incorniciate con un colore diverso. Clicca sulle pagine che vuoi estrarre. Scegli se unire tutte le pagine estratte in un nuovo PDF o se scaricarle in una cartella .zip. Quindi, premi il pulsante Estrai Pagine. Controlla i limiti di 'Estrai Pagine' per ogni tipo di account.

Organizza PDF

Per organizzare le pagine da un PDF, carica il tuo file dal tuo dispositivo o scaricalo direttamente dal cloud tramite il tuo account di Google Drive o Dropbox. Puoi anche rimuovere le pagine dai documenti protetti da password.

Vedrai una miniatura di ogni pagina del tuo file. Quando si caricano più file, le miniature di ciascun file saranno incorniciate con un colore diverso. Quindi puoi rimuovere o trascinare le miniature per riordinare le pagine PDF come preferisci. Premi il pulsante "Organizza PDF" per scaricare il tuo nuovo PDF. Controlla i limiti di 'Organizza PDF' per ogni tipo di account.

Ottimizza PDF

Comprimi PDF

Per comprimere PDF, seleziona i documenti dal tuo dispositivo o dal tuo account di Google Drive o Dropbox. Carica il file o i file da ottimizzare. Scegli tra un livello di compressione estremo, alto o regolare e poi clicca sul tasto 'Comprimi PDF'. Più alta è la compressione, minore la dimensione del file e la sua qualità.

Clicca sul tasto 'Comprimi PDF' per scaricare il tuo PDF compresso oppure salvalo nuovamente nel cloud. Controlla i limiti di 'Comprimi PDF' per ogni tipo di account.

Modifica PDF

Modifica PDF

Per modificare il tuo PDF, carica il tuo file dal tuo dispositivo o dal tuo account di Google Drive o Dropbox.

Una volta all'interno del visualizzatore PDF, utilizza la barra degli strumenti in alto per navigare all'interno del documento. Fai clic sull'icona dell'ingranaggio per scegliere le tue impostazioni della pagina. Puoi visualizzare una o più pagine contemporaneamente. Per spostarti all'interno e intorno alle singole pagine, seleziona l'icona a forma di mano. Utilizza la lente di ingrandimento per ingrandire e ridurre sezioni specifiche del documento.

Per aggiungere elementi o testo a una pagina, scorri verso il basso fino alla pagina che vuoi modificare. Scegli tra le icone Testo, Immagine o Forma disponibili nella barra degli strumenti Modifica PDF nella parte superiore del visualizzatore. Usa il tuo mouse per ridimensionare o spostare qualsiasi elemento come preferisci nella stessa pagina dove è stato creato.

Per aggiungere elementi a una pagina diversa, vai alla pagina e seleziona un nuovo elemento dalla barra degli strumenti. Cambia colore, trasparenza, colore del bordo, rotazione, formato o spessore della linea delle tue immagini, forme o testi usando le loro barre delle proprietà. Fai clic sul pulsante "Modifica PDF" per elaborare il tuo file.

Aggiungere numeri di pagina

Per numerare pagine PDF, carica i tuoi file dal tuo dispositivo o dal tuo account di Google Drive o Dropbox. Scegli se vuoi numerare tutte le pagine o solo le pagine frontali. Scegli l'intervallo di pagina da numerare. Puoi anche modificare la posizione del tuo numero e il suo formato (tipo, dimensione, colore, opacità…) nella barra degli strumenti.

Poi clicca sul tasto 'Aggiungi Numeri Pagina' e scarica o salva di nuovo sul cloud il tuo PDF numerato. Controlla i limiti di 'Numeri Pagina' per ogni tipo di account.

Aggiungere filigrana

Per aggiungere un watermark su un PDF, carica il tuo PDF dal tuo dispositivo o dal tuo account di Google Drive o Dropbox. Puoi combinare testo e immagine nel tuo watermark, che verrà impresso nella stessa posizione in tutte le pagine del tuo file. Puoi personalizzare il formato, l'orientamento e l'opacità.

Poi clicca sul tasto 'Aggiungi Watermark' e scarica o salva di nuovo sul cloud il tuo PDF con il watermark. Controlla i limiti di 'Watermark' per ogni tipo di account.

Ripara PDF

Per riparare un PDF, carica il tuo file PDF danneggiato o corrotto dal tuo dispositivo o dal tuo account di Google Drive o Dropbox. I nostri strumenti cercheranno di analizzare, riparare o recuperare il contenuto dal tuo file corrotto.

Poi clicca sul tasto 'Ripara' e scarica o salva di nuovo sul cloud il tuo PDF riparato. Controlla i limiti di 'Ripara PDF' per ogni tipo di account.

Converti in PDF

JPG a PDF

Per convertire uno o più file da JPG in PDF, carica le tue immagini dal tuo dispositivo o direttamente dal tuo account di Google Drive o Dropbox.

Scegli se vuoi il tuo PDF in uscita in verticale o in orizzontale e se lo vuoi con il margine o senza. Puoi anche unire tutte le immagini in un singolo file PDF. Poi, scarica il tuo PDF o salvalo di nuovo nel cloud. Controlla i limiti di 'JPG in PDF' per ogni tipo di account.

WORD in PDF

Per convertire uno o più documenti da Word in PDF, seleziona i documenti dal tuo dispositivo o dal tuo account del cloud di archiviazione come Google Drive o Dropbox.

Clicca su 'Converti in PDF' per trasformare il tuo .docx in un documento non modificabile. Poi, scarica il tuo PDF oppure salvalo nuovamente nel cloud. Controlla i limiti di 'Word in PDF' per ogni tipo di account.

POWERPOINT in PDF

Per convertire uno o più file da Power Point in PDF, carica i documenti dal tuo dispositivo o direttamente dal Cloud di Google Drive o Dropbox.

Clicca su 'Converti in PDF' per trasformare il tuo .pptx in un documento non modificabile. Tutte le foto e i testi della tua presentazione .pptx saranno convertiti in un file PDF. Poi, scarica il tuo PDF oppure salvalo nuovamente nel cloud. Controlla i limiti di 'Power Point in PDF' per ogni tipo di account.

EXCEL a PDF

Per convertire uno o più file da Excel in PDF, carica i documenti dal tuo dispositivo o direttamente dal tuo account di Google Drive o Dropbox.

Clicca su 'Converti in PDF' per trasformare il tuo .xlsx in un documento non modificabile. Tutte le tabelle e i grafici del tuo Excel saranno convertiti in un documento PDF. Poi, scarica il tuo PDF oppure salvalo nuovamente nel cloud. Controlla i limiti di 'Converti in PDF' per ogni tipo di account.

HTML in PDF

Per convertire una pagina web in PDF, carica le tue immagini dal tuo dispositivo o direttamente dai tuoi account Google Drive o Dropbox.

Incolla l'URL che vuoi convertire nella casella di testo. Verrà mostrata un'anteprima del documento. Puoi personalizzare l'orientamento e il margine del documento risultante dal menu della barra laterale.

Clicca sul tasto'Converti in PDF'. Scarica la tua pagina web esportata o salvala di nuovo sul cloud. Controlla i limiti di 'HTML in PDF' per ogni tipo di account.

Converti da PDF

PDF in JPG

Per convertire un PDF in JPG, seleziona il file dal tuo dispositivo o dal tuo account Google Drive o Dropbox.

Puoi trasformare il tuo PDF in un JPG o estrarre tutte le immagine in un file .zip. Poi, premi sul tasto 'Converti in JPG' per trasformare il tuo PDF in JPG. Controlla i limiti di 'PDF in JPG' per ogni tipo di account.

PDF in WORD

Per convertire uno o più documenti da PDF in Word, seleziona i documenti dal tuo dispositivo o dal tuo account del cloud di archiviazione come Google Drive o Dropbox.

Clicca su 'Converti in Word' per trasformare il tuo PDF in un documento modificabile. Poi, scarica il tuo file .docx oppure salvalo nuovamente nel cloud. Controlla i limiti di 'PDF in Word' per ogni tipo di account.

PDF in POWERPOINT

Per convertire file PDF in Power Point, carica i documenti dal tuo dispositivo o direttamente dal cloud attraverso Google Drive o Dropbox.

Clicca su 'Converti in Power Point' per trasformare il tuo PDF in un documento modificabile. Tutte le immagini, i testi e i grafici del tuo PDF saranno convertiti in un documento Power Point modificabile. Convertendo un documento di più pagine, ogni pagina sarà trasformata in una slide .pptx. Poi, scarica il tuo file .pptx oppure salvalo nuovamente nel cloud. Controlla i limiti di 'PDF in Power Point' per ogni tipo di account.

PDF in EXCEL

Per convertire uno o più file da PDF in Excel, carica i documenti dal tuo dispositivo o direttamente dal tuo account di Google Drive o Dropbox.

Clicca su 'Converti in Excel' per trasformare il tuo PDF in un documento modificabile. Tutte le tabelle e i grafici del tuo PDF saranno convertiti in un documento .xslx. Poi, scarica il tuo file Excel oppure salvalo nuovamente nel cloud. Controlla i limiti di 'PDF in Excel' per ogni tipo di account.

Protezione PDF

Sblocca PDF

Per rimuovere la password da un PDF, carica il documento dal tuo dispositivo o direttamente dal tuo account di Google Drive o Dropbox.

Poi clicca sul tasto 'Sblocca PDF'. Il nostro strumento sbloccherà il tuo documento rimovendo la sua password. Controlla i limiti di 'Sblocca PDF' per ogni tipo di account.

Proteggi PDF

Per proteggere un PDF con una password, carica il documento dal tuo dispositivo o direttamente dal tuo account di Google Drive o Dropbox.

Scegli una password utente per crittografare e proteggere il tuo PDF. Premi poi il tasto 'Proteggi PDF' per aggiungere la password al tuo file. Controlla i limiti di 'Proteggi PDF' per ogni tipo di account.

PDF in PDF/A

Per convertire anche in lotti i tuoi PDF in PDF/A, carica il documento dal tuo dispositivo o direttamente dal tuo account di Google Drive o Dropbox.

Poi clicca sul tasto 'PDF in PDF/A' e scarica o salva di nuovo sul cloud il tuo PDF convertito per l'archiviazione e la conservazione a lungo termine. Controlla i limiti di 'PDF in PDF/A' per ogni tipo di account.

Firma PDF

Per firmare tu stesso il tuo documento

Carica il documento sul nostro strumento Firma PDF. Firma. Scegli di adottare la tua firma preferita dal menu della barra laterale per personalizzare la tua firma. Puoi disegnare la tua firma, caricarne una esistente o firmare con testo. Puoi aggiungere altri campi alla tua firma come la data e le iniziali. Scarica. Fai clic sul tasto 'Firma PDF' per scaricare il tuo PDF firmato o salvarlo di nuovo sul cloud.

Per raccogliere firme elettroniche

Per creare una richiesta di firma, carica il documento sul nostro strumento Firma PDF. Dopo aver caricato il documento PDF, crea una richiesta di firma. Dovrai aggiungere firmatari, includere campi di firma, regolare i parametri di sicurezza e poi inviare. Firma. Tutti i firmatari riceveranno un'email con accesso al documento e saranno in grado di leggerlo e firmarlo. Scaricare. Una volta completato il processo di firma, riceverai una copia del documento firmato insieme all'opzione per scaricare un audit trail. L'audit trail può essere utilizzato come prova nei procedimenti legali.

Concetti tecnologici

File protetti da password

Se elabori un file protetto da password, ti chiederemo di aiutarci ad aprirlo. Una volta che è stato aperto ed elaborato, ci impegnamo a non memorizzare questa password sui nostri server.

Ruotando file

Puoi ruotare i tuoi file passando il mouse su un file e facendo clic sull'icona (). Ma se vuoi ruotare più di 10 file contemporaneamente, puoi farlo automaticamente nella finestra pop-up che apparirà. Prima di poterlo fare, dovrai eseguire il rendering di tutte le pagine per ruotarle nella stessa direzione.

Rotazione JPG in PDF

La ruotazione di un'immagine in un documento PDF influisce solo sull'immagine. In seguito potrai scegliere l'orientamento della pagina (verticale od orizzantale) con l'immagine appena ruotata.

Organizzando file

I file possono essere organizzati in varie maniere. Trascinali e rilasciali come preferisci; puoi anche ordinarli alfabeticamente cliccando sull'icona di (); e hai anche la possibilità di ordinarli secondo l'alfabeto al contrario cliccando sull'icona ().

Comprimendo file

Dopo aver elaborato i tuoi file, vedrai la dimensione del file originale e in quale percentuale sono stati ridotti. Nel caso di file PDF già ottimizzati, non saremo in grado di comprimerli ulteriormente.

Caricando ed elaborando

Il tempo di caricamento e di elaborazione dei file dipenderanno dalla tua connessione internet e dalla dimensione dei file che selezioni. Se il browser si chiude durante l'elaborazione dei tuoi file, puoi verificare lo stato dell'elaborazione nel menu 'Il Mio Account', nella sezione 'Ultimi file elaborati'.

Scaricando

I file in uscita verranno cancellati dai nostri server dopo un periodo di tempo, a seconda del tipo di utente. Puoi anche salverli direttamente nel cloud, nel tuo account di Google Drive o Dropbox. Puoi annullare la validità del link di download quando vuoi.

Lavorare dal cloud

Non hai bisogno di avere i file che vuoi elaborare sul tuo dispositivo. Se hai un Account di Google Drive o Dropbox, puoi scaricarli direttamente sulla nostra pagina. Una volta che i tuoi file sono stati elaborati. puoi salvarli nuovamente nel cloud. Per elaborare i file archiviati nel cloud, devi inserire la password del tuo account del cloud la prima volta che lo utilizzi. Le tue password non saranno mai memorizzate nel nostro sistema. Questa caratteristica è particolarmente utile lavorando da un dispositivo mobile o da un tablet.

Seleziona i file da caricare

Per selezionare i file dal tuo computer, puoi cliccare sul tasto 'Seleziona immagini'. Se vuoi selezionare più di un file allo stesso tempo, tieni premuto il tasto 'Ctrl' mentre fai clic sui file. Se i file sono in ordine consecutivo, fai clic sul primo e sull'ultimo, tenendo premuto il tasto 'Shift' Puoi anche trascinarli direttamente sul nostro spazio di lavoro.

Requisiti minimi

Consigliamo di utilizzare i nostri strumenti dai seguenti browser: Chrome, Firefox, Explorer +10 e Safari. Lavorando da Explorer versione 8 e 9, l'opzione di rendering per le pagine del documento sarà disabilitata. iLovePDF ha bisogno di lavorare con Javascript abilitato.

Problemi tecnici

Questa nuova versione di iLovePDF è stata progettata per essere molto più semplice da usare, e abbiamo cercato di risolvere qualunque problema tecnico. Tuttavia, se dovessi trovare qualsiasi problema, non esitare a farcelo sapere. Se il file non contiene materiale confidenziale, allegalo alla tua email così potremo risolvere il problema più velocemente. Faremo tutto il possibile per trovare una soluzione.
